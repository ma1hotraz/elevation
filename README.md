# Kulkaran Coaching Institute — “Learn Better” Landing Page

A conversion-focused Next.js landing page recreated from the selected **Learn Better. Achieve Bigger.** design direction.

## Included

- Founder-led hero section
- Hero enquiry form
- Offline PCM, IELTS Hybrid, and French Online program cards
- Why choose us section
- Founder section
- Testimonials/results
- FAQ
- Final CTA and footer
- CSS-only micro animations
- `prefers-reduced-motion` support
- No external fonts or animation libraries

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Replace before launch

Update these placeholders in `app/page.tsx`:

- Phone number
- Email address
- Location
- Student/result numbers if needed
- Form backend/action when ready
